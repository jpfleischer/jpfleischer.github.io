Cloudmesh Toolkit for MLCommons Satellite Image and Earthquake Analysis


PI Names: Dr. Gregor von Laszewski and Dr. Geoffrey C. Fox

Institution and Department: University of Virginia, Biocomplexity Institute

Program: Computing for Global Challenges

Affiliation Duration: May 2022 - Jan 2024

NSF Award: 1829704

Research Focus

This project used a Python toolkit to accomplish and streamline several deep learning applications, such as finding the best model for a satellite classifier and an earthquake forecaster. This analysis was done as part of MLCommons, which is an open-source initiative and standard used to compare models across many industries and datasets.

Project Responsibilities

I leveraged the Rivanna supercomputer to conduct geospatial earthquake forecasting using LSTM and TFT. I also designed hybrid analytics workflow software for the open-source Cloudmesh project.
